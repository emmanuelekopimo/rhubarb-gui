# Rhubarb Lip Sync GUI - Release Notes

## v1.1.0

**Release Date:** February 16, 2026

### What's New

Enhanced GUI with audio and lip sync preview capabilities, plus stability improvements.

### New Features

- **Audio Preview** - Listen to selected audio files before processing
- **Lip Sync Preview** - Watch animated lip movements synchronized with audio playback
- **Mouth Shape Visualization** - Real-time display of mouth animations during playback
- **Format Parsing** - Automatic parsing of TSV, JSON, XML, and DAT output formats
- Support for seeking/scrubbing through audio during preview
- Play, pause, and stop controls for audio preview

### Bug Fixes & Improvements

- Removed "Quiet mode" option that caused app misbehavior
- Machine-readable output now always enabled (required for progress tracking)
- Improved window geometry and layout
- Better command preview display with adjusted margins and alignment
- Enhanced UI responsiveness

### Features

- Easy-to-use GUI with tabbed interface
- Real-time progress tracking
- Support for multiple output formats (TSV, XML, JSON, DAT)
- Command preview before execution
- Flexible speech recognition options
- Audio and lip sync preview with mouth animation

## System Requirements

- Windows 10 or Windows 11
- ~200MB disk space

## Installation

Run the installer and follow the prompts. Everything is included - just install and use.

## Quick Start

1. Open the application
2. Select an audio file (WAV or OGG)
3. Choose output file location
4. Click "Run Rhubarb Lip Sync"
5. Use "Preview Audio" to listen to the file, or "Preview with Lip Sync" to see mouth animation (after processing)

See README.md for more details.
